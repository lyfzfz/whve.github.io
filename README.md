我,没什么特别的,习惯不好,曾坚信有短时速效药改变自己,事实证明,自己没有所预想的自制力,不经意间就会回到习惯的轨道.现在在学习之余,想做一些自己喜欢的事.比如,写些文字,学习些语言.

20多年前,来自中国一个美丽的乡村,如今背井离乡,在外求学.
